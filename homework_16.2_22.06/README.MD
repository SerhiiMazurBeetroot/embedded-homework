# Бонусне завдання: плавний нічник на ESP32 (LEDC ШІМ)

Мета: модернізувати звичайне двопозиційне керування світлом (увімкнено/вимкнено) до системи з плавною зміною яскравості. Наш пристрій має працювати як інтелектуальний нічник: чим темніше в кімнаті, тим яскравіше світиться LED-діод, і навпаки.

Опис логіки роботи:
замість того, щоб просто вмикати світло при переході через певне порогове значення, ми будемо динамічно масштабувати рівень освітленості з фоторезистора (вхід) у рівень яскравості світлодіода (вихід).

Підказка: як працює функція `map()`?
У цьому завданні вам доведеться працювати з двома абсолютно різними модулями мікроконтролера, які мають різну розрядність (бітність):

Вхід (АЦП / ADC): Має розрядність 12 біт. Тобто приймає значення від `0` (повна темрява) до `4095` (максимальне світло).
Вихід (ШІМ / PWM): У коді ми налаштували його на 8 біт. Тобто він приймає значення від `0` (вимкнено) до `255` (максимальна яскравість).
Щоб красиво та пропорційно перевести значення з однієї шкали в іншу, в Arduino IDE є вбудована математична функція map().

long result = map(value, fromLow, fromHigh, toLow, toHigh);

value — поточне значення, яке ми хочемо змінити.
fromLow / fromHigh — мінімальна та максимальна межі початкового діапазону.
toLow / toHigh — мінімальна та максимальна межі нового діапазону.

Важливий трюк для нічника (Інверсія): зазвичай функція масштабує прямо пропорційно (наприклад, від 0..4095 до 0..255). Але нам потрібно навпаки: щоб при мінімальному світлі (0 на АЦП) світлодіод горів максимально яскраво (255 на ШІМ).
Для цього ми просто міняємо місцями кінцеві значення у функції: map(adc, 0, 4095, 255, 0);
